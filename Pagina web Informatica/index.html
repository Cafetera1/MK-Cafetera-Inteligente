Como manejar la cafetera

En la cafetera podras encontrar botones para hacer manualmente el cafe, habra botones para encender, apagar, el tipo de cafe que quieres y empezar el preparado de cafe. Primero debes presionar el boton de "encender", luego el de "preparar cafe", esperamos a que este listo el cafe y una vez que este listo el cafe, apagamos la cafetera.
